Resume

tanpa konflik

  • membuat branch baru dengan menhetikkan "git branch (nama branch)" pada terminal
  • setelah membuat branch baru, pindah dari branch master dengan mengetikkan "git checkout (nama branch) pada terminal"
  • untuk mengecek apakah sudah berada pada branch yang di tuju, ketikkan "git branch" pada terminal
  • lanjut tambahkan code yang ingin di tambahkan, setelah itu lakukan langkah-langkah git push
  • setelah itu lakukan merge dari branch master, caranya pindah dulu ke branch master dengan cara menggunakan perintah "git checkout (nama branch)" pada terminal

Resume dengan konflik

  1. ikuti langkah di atas
  2. membuat branch baru
  3. menambahkan code, lakukan sampai ke tahap push
  4. pindah ke branch master
  5. lakukan perubahan pada line yang sama dengan branch baru
  6. lakukan hingga git push
  7. git merge
